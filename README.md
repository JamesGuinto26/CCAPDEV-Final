# Eatstagram: Restaurant Review Application

This project attempts to recreate a restaurant review web application with the use of HTML, Javascript, Mongoose, Handlebars, and Express. Registered users here are able to give reviews of the restaurants available in the application by providing ratings, comments, or even images.   

# Features

A registered user in the application can do the following:

- Select a restaurant and create a review of that specific resto
- View detailed information about each restaurant, including address, contact info, operating hours, and overall ratings.
- Access a list of past reviews left by other users as well as filtering reviews by rating
- See comments left by restaurant owners or admins on specific reviews
- Edit and delete their own reviews on the web application
- Viewing another user's profile and viewing their reviews
- Create and edit their own user profile for others users to see
- Mark a review as helpful or unhelpful for other users to see
- Admins or establishment owners are able to leave a comment on reviews on their managed restaurant
